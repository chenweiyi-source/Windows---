## 6. AI 辅助测试实践

在开发过程中，使用 Claude Code 辅助生成了代码实现，但**测试用例全部由人工编写**，遵循以下原则：

1. **测试先行**：每个功能的测试用例在编码前编写，明确验收标准
2. **边界覆盖**：每个功能除了正向用例，还包括至少一个边界或异常场景用例
3. **人工验证**：所有测试结果均为人工逐项操作验证，不依赖自动化测试框架
4. **测试驱动 AI**：测试用例中定义的验收标准直接作为 Prompt 设计的一部分输入给 Claude Code，确保生成的代码有明确的目标
