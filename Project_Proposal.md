# Project Proposal — DailyNote 日记本应用

## 1. 项目概述与目标

DailyNote 是一款基于 Windows Forms 的桌面日记本应用，旨在为用户提供简洁、美观、易用的日记记录与管理体验。用户可以通过该应用新建日记、浏览日记列表、编辑已有日记、按关键字搜索日记，所有数据以 JSON 格式持久化存储在本地。

**核心目标：**
- 运用 WinForm 控件编程与 GDI+ 自绘技术构建精致用户界面
- 实现完整的日记 CRUD（增删改查）功能
- 通过 JSON 文件实现数据持久化
- 以 AI 辅助编程方式探索 RDD（需求驱动开发）模式

## 2. 功能需求与用户故事

| 编号 | 功能 | 用户故事 |
|------|------|----------|
| F1 | 新建日记 | 作为用户，我想要填写标题、正文、选择日期和心情标签，以便记录当天的心情和事件 |
| F2 | 浏览日记列表 | 作为用户，我想要按日期倒序查看所有日记，每项显示日期、标题和心情，以便快速浏览 |
| F3 | 编辑日记 | 作为用户，我点击列表中的日记项可以加载到编辑区进行修改并保存 |
| F4 | 删除日记 | 作为用户，我可以删除不再需要的日记，删除前有确认提示防止误操作 |
| F5 | 搜索日记 | 作为用户，我在搜索框输入关键字可以实时过滤日记列表，按标题或正文匹配 |
| F6 | 数据持久化 | 作为用户，我关闭程序后再次打开时，之前保存的日记不会丢失 |

## 4. 系统架构设计

```
┌─────────────────────────────────────────────────────┐
│                    表现层 (UI)                        │
│  Form1 ── 左侧 Panel (侧边栏) ── 右侧 Panel (编辑区) │
│    ├── ListBox (日记列表, OwnerDraw)                 │
│    ├── TextBox (搜索框, 标题, 正文)                  │
│    ├── DateTimePicker (日期选择)                     │
│    ├── ComboBox (心情选择)                           │
│    └── Button (新建/保存/删除/清空)                  │
├─────────────────────────────────────────────────────┤
│                    业务逻辑层                         │
│    ├── NewEntry / SaveEntry / DeleteEntry            │
│    ├── RefreshList / LoadEntry                       │
│    └── 搜索过滤逻辑 (LINQ)                            │
├─────────────────────────────────────────────────────┤
│                    数据层                             │
│    ├── DiaryEntry 数据模型                            │
│    ├── LoadData() — JSON 反序列化                      │
│    └── SaveData() — JSON 序列化                       │
└─────────────────────────────────────────────────────┘
```

**分层说明：**

- **表现层**：负责所有 UI 布局和用户交互，所有控件通过纯代码方式创建和配置
- **业务逻辑层**：处理日记的增删改查、搜索过滤、表单状态管理
- **数据层**：定义 DiaryEntry 数据模型，通过 System.Text.Json 实现数据读写

## 5. 模块功能说明

### 5.1 数据模型模块

定义 `DiaryEntry` 类，包含 6 个字段：
- `Id` — GUID 前 8 位，全局唯一标识
- `Title` — 日记标题
- `Content` — 日记正文
- `Mood` — 心情标签（开心/一般/难过/兴奋/疲惫）
- `Date` — 日记日期（默认为当天）
- `CreatedAt` — 创建时间戳

### 5.2 数据持久化模块

- 使用 `System.Text.Json` 将 `List<DiaryEntry>` 序列化为 JSON
- 存储于程序运行目录下的 `diaries.json`
- 程序启动时自动加载 (`LoadData`)，修改后自动保存 (`SaveData`)
- 完整 try-catch 异常处理，加载失败时初始化为空列表

### 5.3 日记列表展示模块

- ListBox 设置为 `OwnerDrawVariable` 模式
- 每项显示日期（8pt）、标题（10pt加粗，超长截断）、心情标签
- 隔行交替背景色，选中行使用主题色高亮
- 按日期倒序排列

### 5.4 日记编辑模块

- 右侧编辑面板包含标题输入、正文编辑、日期选择和心情下拉框
- 通过 `_isEditing` 标志区分新建/编辑模式
- 标题非空校验、删除确认对话框

### 5.5 搜索模块

- 基于 `TextChanged` 事件实时触发过滤
- LINQ `Where` + `string.Contains` 模糊匹配
- 忽略大小写，关键字为空时显示全部
